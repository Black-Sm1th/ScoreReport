# ScoreReport

## 新功能：简化版OCR图片识别

### 功能介绍
新增的简化版OCR功能允许用户通过右键菜单读取本地测试图片并进行模拟识别，用于测试完整的OCR工作流程。

### 主要特性
- 🖼️ **一键测试**：右键菜单选择"截图识别"即可读取测试图片
- 🎯 **无依赖**：不需要安装复杂的Tesseract OCR库
- 📋 **便捷复制**：识别结果可直接复制到剪贴板
- ✏️ **文本编辑**：支持在结果对话框中编辑识别的文字
- 🔧 **自动创建**：自动创建包含测试文字的图片文件

### 使用方法
1. 右键点击悬浮窗
2. 选择"截图识别"选项
3. 系统自动读取或创建测试图片
4. 在弹出的对话框中查看模拟识别结果
5. 点击"复制到剪贴板"保存结果

### 技术特点
- ✅ **即开即用**：无需额外安装任何依赖
- ✅ **完整流程**：测试从图片读取到结果显示的完整流程
- ✅ **稳定可靠**：不会出现库初始化失败等问题
- ✅ **易于扩展**：为将来集成真实OCR功能做好准备
